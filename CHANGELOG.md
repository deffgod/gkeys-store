# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-01-09

### Added
- Полная документация проекта
- Скрипты для настройки окружения (`setup-env.ts`)
- Скрипты для синхронизации базы данных (`sync-db.ts`)
- `.env.example` файлы для frontend и backend
- Документация по миграциям Prisma
- Quick Start Guide для новой команды
- Project Handover документ

### Changed
- Обновлен `.gitignore` для исключения ненужных файлов
- Очищены старые бэкапы базы данных
- Улучшена структура документации

### Fixed
- Удалены временные и бэкап файлы
- Обновлена документация по переменным окружения

### Security
- Все секретные ключи теперь только в `.env` файлах (не в репозитории)
- Добавлены `.env.example` файлы с примерами конфигурации

## Типы изменений

- `Added` - для новых функций
- `Changed` - для изменений в существующей функциональности
- `Deprecated` - для функций, которые скоро будут удалены
- `Removed` - для удаленных функций
- `Fixed` - для исправления ошибок
